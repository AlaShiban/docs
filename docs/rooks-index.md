# {{ page.title}}

Rooks are the component that allows you to collect data directly from a running application.

Information about the Rook's configuration can be found [here](/rooks/config).

## Supported Platforms:
- [Python](/rooks/python)
- [JVM](/rooks/java)
- [NodeJS](/rooks/node)
- .Net
- Ruby
- Go
